## 概览

本次更新新增 **4 个核心模块**，增强了数据集质量管理和模型训练优化能力。

### 新增模块

| 模块 | 命令 | 功能 |
|------|------|------|
| Anchor Generator | `yolo-anchors` | K-means 聚类生成最优 anchors |
| Hyperparameter Tuner | `yolo-tune` | 遗传算法超参数进化调优 |
| Label QC | `yolo-label-qc` | 标签质量检查与自动修复 |
| Pipeline Orchestrator | `yolo-pipeline` | 全流程串联编排 |

## 新增命令详解

### 1. yolo-anchors - Anchor 生成工具

**功能:** 从数据集自动生成最优 anchor boxes

**核心特性:**
- K-means 聚类 (IoU-based distance)
- 自动选择最优 k 值 (轮廓系数)
- 多尺度 anchors (P3/P4/P5)
- 可选验证对比 mAP

**使用方式:**
```bash
# 生成 anchors
yolo-anchors --data dataset/ --output anchors.yaml

# 生成并验证
yolo-anchors --data dataset/ --output anchors.yaml --validate --model best.pt

# 自定义 k 范围
yolo-anchors --data dataset/ --k_min 6 --k_max 12
```

**相关文件:**
- `src/tools/anchor_generator.py`
- `tests/tools/test_anchor_generator.py`

---

### 2. yolo-tune - 超参数调优工具

**功能:** 使用遗传算法自动搜索最优超参数

**核心特性:**
- 封装 YOLO 内置 hyperparameter evolution
- 全面调优 (lr/momentum/weight_decay/augmentation 等)
- 10 代进化 (可配置)
- 生成最优超参数 YAML

**使用方式:**
```bash
# 基础用法
yolo-tune --data dataset.yaml --model yolo11n.pt

# 自定义代数
yolo-tune --data dataset.yaml --model best.pt --generations 20 --epochs 5

# 指定输出
yolo-tune --data dataset.yaml --output tuned_params.yaml
```

**相关文件:**
- `src/tools/hyperparameter_tuner.py`
- `tests/tools/test_hyperparameter_tuner.py`

---

### 3. yolo-label-qc - 标签质量检查工具

**功能:** 检测并自动修复标签质量问题

**检测问题类型:**

| 问题 | 检测 | 自动修复 |
|------|------|---------|
| 重复框 (IoU > 0.9) | ✅ | ✅ 删除重复 |
| 过小 bbox (< 0.05%) | ✅ | ✅ 删除 |
| 过大 bbox (> 95%) | ✅ | ❌ 仅报告 |
| 遮挡 (像素方差低) | ✅ | ❌ 仅报告 |

**使用方式:**
```bash
# 仅检查不修复
yolo-label-qc --data dataset/

# 检查并自动修复
yolo-label-qc --data dataset/ --auto_fix

# 自定义阈值
yolo-label-qc --data dataset/ --auto_fix --iou_threshold 0.85 --min_area 0.001
```

**安全措施:**
- 修复前自动备份到 `labels_backup/`
- 支持 `--dry-run` 预览

**相关文件:**
- `src/tools/label_qc.py`
- `tests/tools/test_label_qc.py`

---

### 4. yolo-pipeline - 流程编排工具

**功能:** 通过 YAML 配置串联多个工具

**核心特性:**
- YAML 配置定义完整流程
- 支持条件执行
- 失败策略 (stop/skip)
- 执行报告生成

**YAML 配置示例:**
```yaml
name: "完整训练流程"

stages:
  - name: "数据校验"
    tool: "validate"
    params:
      input: "./dataset"
      fix: true

  - name: "标签质量检查"
    tool: "label-qc"
    params:
      data: "./dataset"
      auto_fix: true

  - name: "生成 Anchors"
    tool: "anchors"
    params:
      data: "./dataset"
      output: "./anchors.yaml"

  - name: "训练模型"
    tool: "train"
    params:
      data: "./dataset.yaml"
      epochs: 100
      device: "0"

  - name: "导出模型"
    tool: "export"
    params:
      format: "onnx"
```

**使用方式:**
```bash
# 执行 pipeline
yolo-pipeline --config pipeline_train.yaml

# 仅预览不执行
yolo-pipeline --config pipeline_train.yaml --dry-run

# 覆盖参数
yolo-pipeline --config pipeline_train.yaml --set epochs=50
```

**相关文件:**
- `src/tools/pipeline.py`
- `tests/tools/test_pipeline.py`

## 快速开始

### 完整训练流程

```bash
# 1. 下载模型
yolo-download --model yolo11n

# 2. 数据校验
yolo-validate --input dataset/ --fix

# 3. 标签质量检查
yolo-label-qc --data dataset/ --auto_fix

# 4. 生成 anchors
yolo-anchors --data dataset/ --output anchors.yaml

# 5. 超参数调优 (可选，耗时)
yolo-tune --data dataset.yaml --model yolo11n.pt --generations 10

# 6. 训练
yolo-train --data dataset.yaml --epochs 100

# 7. 导出
yolo-export --model runs/train/exp/weights/best.pt --format onnx
```

### 使用 Pipeline (推荐)

```yaml
# pipeline_train.yaml
name: "完整训练流程"
stages:
  - name: "数据校验"
    tool: "validate"
    params:
      input: "./dataset"
      fix: true
  - name: "训练"
    tool: "train"
    params:
      data: "./dataset.yaml"
      epochs: 100
```

```bash
yolo-pipeline --config pipeline_train.yaml
```
